# ⚽ FIFA World Cup 2026 Performance Dashboard

## 📌 Project Overview
This project is an interactive football analytics dashboard designed to analyze FIFA World Cup 2026 team and player performance. The dashboard provides key performance indicators (KPIs), player statistics, and match insights using dynamic filters and visualizations.

Although developed in **Microsoft Excel**, the dashboard follows **Power BI dashboard design principles**, including KPI cards, interactive slicers, and data-driven visualizations.

## 🎯 Objectives
- Analyze overall team performance
- Identify top goal scorers
- Compare goals scored by team and opponents
- Track wins, losses, and draws
- Monitor defensive performance (blocks & saves)
- Visualize disciplinary records (yellow/red cards)

## 📊 Dashboard Features
- Interactive Team & Player Filters
- KPI Cards
  - Matches Played
  - Matches Won
  - Matches Lost
  - Matches Drawn
  - Average Performance Score
- Top Goal Scorers
- Goals Comparison
- Blocks & Saves Analysis
- Card Distribution
- Performance Gauge
